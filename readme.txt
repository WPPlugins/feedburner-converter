With this plugins you can rewrite default wordpress feeds by others like as feedburner. This plugin is compatible with qTranslate plugin and you can rewrite feeds for all defined languages.

== Description ==

Now you can add feedburner to your wordpress template in easy way. The only that you do is add your new link after old.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in options page.
